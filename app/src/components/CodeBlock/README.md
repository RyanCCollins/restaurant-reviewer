## CodeBlock Component
A component that shows as a code block.

### Example

```js
<CodeBlock
  codeBlock={
    `Anything that goes here will be shown as code`
  }
/>
```

### Props

| Prop          | Type     | Default     | Possible Values
| ------------- | -------- | ----------- | ---------------------------------------------
| **codeBlock**    | String   |             | Any string value to be shown as code in a code block.
