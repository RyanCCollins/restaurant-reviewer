## AddButton Component
A component that acts as a button to add items.  Contains a + icon.

### Example

```js
<AddButton onAdd={this.handleAdd} />
```

### Props

| Prop          | Type     | Default     | Possible Values
| ------------- | -------- | ----------- | ---------------------------------------------
| **onAdd**    | Func   |             | A callback for when the button is clicked.
